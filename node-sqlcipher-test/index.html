sqlcipher 测试

Node.js , Chromium , and Electron .

把sqlcipher编译为electron模块

命令：

npm install --save "@journeyapps/sqlcipher"

cd node_modules\@journeyapps\sqlcipher

node-pre-gyp rebuild --runtime=electron --target=9.3.5 --msvs_version=2015 --arch=ia32 --disturl=https://atom.io/download/atom-shell --abi=80
